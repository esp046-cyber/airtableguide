Airtable Fundamentals

Understanding the Structure

Workspace: Your organization's container for all bases

Base: Like a database - contains multiple tables

Table: Like a spreadsheet - contains records and fields

Record: A row in your table (one item)

Field: A column in your table (a property)

Creating Your First Base

1 Click "Create a base" from your workspace dashboard
2 Choose a method: Start from scratch, use a template, or import data
3 Name your base something descriptive (e.g., "Marketing Campaign 2026")
4 Add your first table and start building your structure

Setting Up Automations

1 Click "Automations" in the top toolbar
2 Choose a trigger: When record created, updated, or matches conditions
3 Add actions: Send email, update record, create record, or run script
4 Test and activate your automation
